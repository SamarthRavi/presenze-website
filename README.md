# Presenze Website

Official landing page for **Presenze** — a modern Bluetooth Low Energy (BLE) based attendance platform built for educational institutions.

The website showcases the Presenze ecosystem, including the Faculty App, Student App, Dashboard, core features, product video, and company information.

🌐 **Live Website:** https://presenze.website

---

## About

Presenze is designed to modernize attendance management using secure Bluetooth Low Energy technology.

The website serves as the official product landing page and provides:

- Product Overview
- Feature Showcase
- Product Introduction Video
- Screenshots
- Timeline
- FAQ
- Founder Information
- Download Links
- Privacy Policy
- Terms & Conditions

---

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Vercel

## Installation

Clone the repository

```bash
git clone https://github.com/SamarthRavi/presenze-website.git
```

Navigate to the project

```bash
cd presenze-website
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```

---

## Production Build

```bash
npm run build
```

Start production server

```bash
npm start
```
